# UC15 Lavanderia - Front-end (HTML/CSS/JS)

Entrega da etapa de front-end do PI (sem backend).

## Páginas
- `login.html` (validação CPF + matrícula)
- `dashboard.html` (KPIs simulados)
- `clientes.html` (CRUD simulado via LocalStorage)
- `operadores.html` (CRUD simulado via LocalStorage)

## Organização
- `css/styles.css`
- `js/app.js` (tema claro/escuro)
- `js/login.js` (validação do login)
- `js/dashboard.js` (KPIs + logout)
- `js/crud.js` (CRUD e validações)
