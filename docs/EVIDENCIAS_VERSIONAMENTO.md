# Evidências de Versionamento (Git)

## Repositório
Crie um novo repositório no GitHub, por exemplo: `uc15-lavanderia-web-spring`.

## Commits sugeridos (ordem)
1. `feat: criar projeto Spring Boot com dependências web e JPA`
2. `feat: implementar entidades e repositórios JPA (Cliente, Operador)`
3. `feat: implementar serviços com validações e regras de negócio`
4. `feat: criar controllers REST e endpoints de autenticação`
5. `chore: integrar front-end em resources/static`
6. `test: adicionar testes (CpfUtils + contextLoads)`
7. `docs: adicionar evidências e instruções de execução`

## Prints para evidência
- Print do GitHub mostrando os arquivos do projeto
- Print do histórico de commits (com as mensagens acima)
- Print do build/testes (mvn test) com sucesso
